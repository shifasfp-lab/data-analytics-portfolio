# Unicorn Startup Analysis Dashboard (Power BI)

## Tools Used
Power BI  
Data Modeling  
Data Visualization  

## Project Overview
This project analyzes the global unicorn startup ecosystem,
including venture capital funding, investors, industries,
and geographic distribution of startups.

## Key Insights
- Unicorn startups increased rapidly after 2015
- Fintech and Internet Software attract the highest funding
- Major startup hubs appear in North America and Asia
- Certain investors repeatedly back successful startups
