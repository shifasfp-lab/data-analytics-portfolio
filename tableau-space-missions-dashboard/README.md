# Space Missions Data Visualization Dashboard (Tableau)

## Tools Used
Tableau  
Data Visualization  
Exploratory Data Analysis  

## Project Overview
This project analyzes global space missions from 1958 to 2021 using an interactive Tableau dashboard.

The analysis explores launch trends, mission success rates, leading space agencies, and geographic distribution of launches.

## Key Insights
- Space missions increased significantly during the space race era
- USA and Russia dominate global launch activity
- Mission success rates improved dramatically over time
- Modern space exploration shows growth driven by private companies
